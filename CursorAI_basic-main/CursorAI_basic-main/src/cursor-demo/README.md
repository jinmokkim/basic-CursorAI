# cursor-demo

사용자 배열에서 이메일을 추출·검증·중복 제거하는 유틸리티 모듈입니다.

## 릴리스 노트

### v1.0.0

사용자 배열에서 이메일을 추출·검증·중복 제거하는 유틸리티 모듈을 추가했습니다.

#### ✨ 기능

- `extractEmails` — 사용자 객체 배열에서 이메일 필드를 추출합니다. 배열이 아닌 입력은 빈 배열을 반환합니다.
- `isValidEmail` — RFC 5322 정규식과 RFC 3696 길이 제한(로컬 파트 64자, 전체 254자)으로 이메일 형식을 검증합니다.
- `getValidEmails` — 사용자 배열에서 유효한 이메일만 필터링합니다.
- `uniqueValidEmails` — 유효한 이메일을 추출하고 중복을 제거합니다.
