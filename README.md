# basic-CursorAI

Cursor AI 기본 강의 실습 자료 저장소입니다.

## 포함 내용

### 실습 프로젝트

- **cursor-demo** — 이메일 검증·로그인 API Node.js 실습 (`src/cursor-demo/`)
- **tetris-cursor** — HTML/CSS/JS 테트리스 게임 실습 (`src/tetris-cursor/`)
